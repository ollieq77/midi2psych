MIDI to Psych Engine Converter is a tool that transforms MIDI files into chart files compatible with Friday Night Funkin' Psych Engine. It features both a graphical interface for easy use and a command-line interface for advanced users and automation.

‚ú® Key Features

  • Dual Interface: GUI for beginners, CLI for power users
  • Two-Track Support: Separate MIDI files for Player 1 and Player 2
  • Tempo Change Detection: Automatically handles BPM changes
  • Sustain Notes: Optional support for hold notes
  • Customizable Output: Configure song metadata, characters, and timing
  • High Precision: Adjustable decimal precision for note timing
  • Real-time Progress: Visual feedback during conversion

üöÄ Getting Started

GUI Mode (Recommended for Beginners)

  1. Double-click midi2psych.exe to launch the GUI
  2. Click "Browse..." to select your Player 1 MIDI file
  3. Click "Browse..." to select your Player 2 MIDI file
  4. Choose where to save the output JSON file
  5. Configure your chart settings (optional)
  6. Click "CONVERT" and wait for completion

CLI Mode (For Advanced Users)

Open Command Prompt or Terminal in the folder containing midi2psych.exe and run:

midi2psych.exe player1.mid player2.mid output.json [options]

üéπ MIDI Preparation Tips

üí° Getting the Best Results

  • Note Mapping: The converter uses note number modulo 4 for lanes (0-3 = left, down, up, right)
  • Separate Tracks: Use separate MIDI files for each player's chart
  • Clean Your MIDI: Remove unnecessary tracks, controller data, and silence
  • Tempo Map: Include tempo changes in your MIDI if the song changes speed
  • Test in DAW: Preview your MIDI in a DAW to verify it sounds correct before converting

‚ö†Ô∏è Common Issues

  • Missing Notes: Check your Min Velocity setting if notes are missing
  • Wrong Timing: Adjust the Offset setting if all notes are early/late
  • Wrong Speed: Use BPM Multiplier if the chart plays too fast or slow
  • File Won't Load: Make sure your MIDI files are valid and not corrupted
  • Notes Are Different Types: Make sure all your notes are on Track 1

üìÇ Output Format

The converter generates a JSON file compatible with Psych Engine. To use it:

  1. Place the JSON file in your Psych Engine mod's data/[songname]/ folder
  2. Name it according to the difficulty (e.g., [songname]-hard.json, [songname].json, [songname]-easy.json)
  3. Make sure you have the corresponding audio files in songs/[songname]/

‚ùì FAQ

Can I use the same MIDI for both players?

Yes, but it will duplicate the chart. For best results, create separate MIDI files with different patterns for each player.

Why are some notes missing?

This is usually due to the Min Velocity filter. MIDI notes with velocity below your threshold are ignored. Try setting Min Velocity to 0.

Can I convert multiple difficulties at once?

Not directly, but you can run the converter multiple times with different settings (like BPM multiplier or note offset) to create multiple difficulty levels.

Does this work with other FNF engines?

It's specifically designed for Psych Engine. Other engines may have different JSON formats and might not work correctly.

How do I map specific MIDI notes to specific lanes?

The converter uses note % 4 to determine lanes. So MIDI notes 0, 4, 8, 12... map to lane 0 (left), notes 1, 5, 9, 13... map to lane 1 (down), etc.

üìù Technical Details

How It Works

  1. Parse MIDI: Reads both MIDI files and extracts note events and tempo changes
  2. Convert Timing: Converts MIDI ticks to milliseconds using PPQ and BPM data
  3. Map Notes: Maps MIDI note numbers to FNF lanes (0-3 for P1, 4-7 for P2)
  4. Create Sections: Divides the chart into sections based on timing and BPM changes
  5. Determine Must-Hit: Decides which sections are "must hit" based on note distribution
  6. Generate JSON: Outputs a Psych Engine compatible JSON structure
